##  Описание проекта

**Advanced Currency Converter** — это современное, лёгкое и полностью адаптивное веб‑приложение, созданное без фреймворков.  
Функциональность включает:

-  Конвертацию валют в реальном времени  
-  Поиск валюты по стране  
-  Историю операций (localStorage)  
-  Экспорт истории в CSV  
-  Сохранение пользовательских настроек  
-  Тёмную/светлую тему  
-  График курса (Chart.js)  
-  Mobile‑first дизайн

##  Логические модули

###  1. API‑модуль (внутри `app.js`)
- Загрузка курсов валют с `open.er-api.com`
- Поддержка альтернативных API
- Обработка ошибок сети

###  2. Конвертер
- Автоматический пересчёт
- Форматирование чисел
- Обновление UI и графика

###  3. История (localStorage)
- Сохранение последних 50 операций
- Рендер списка
- Очистка
- Экспорт в CSV

###  4. Темизация
- CSS‑переменные
- Переключатель темы
- Хранение выбора в localStorage

###  5. Поиск валюты по стране
- Быстрый поиск по справочнику
- Автоматическое обновление селектора валюты

###  6. График курса (Chart.js)
- Псевдо‑данные за 7 дней
- Градиентная заливка
- Готовность к подключению реального API

## 🚀 Локальный запуск

### Вариант 1 — открыть файл напрямую
Открыть `index.html` в браузере.  
Если браузер блокирует `fetch`, используй локальный сервер.

### Вариант 2 — локальный сервер (рекомендуется)

**Python 3**
```bash
python -m http.server 8000
```

**Node.js**
```bash
npx serve .
```

Открыть:  
`http://localhost:8000`
